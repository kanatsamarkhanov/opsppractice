# 🏭 Индустриялық Қазақстан — Интерактивті практикалық сабақ

**2-курс география педагогтары студенттеріне арналған**  
**2 сағатта аяқталатын цифрлық практика**

---

## 🚀 Streamlit Cloud-қа жариялау (5 минут)

1. `app.py` және `requirements.txt` файлдарын GitHub репозиторийге жүктеңіз
2. [share.streamlit.io](https://share.streamlit.io) → **New app**
3. Репозиторийді таңдап, `app.py` көрсетіңіз
4. **Deploy** басыңыз → сілтеме дайын

---

## 📚 Сабақ құрылымы (2 сағат)

| Бөлім | Уақыт | Мазмұн |

## 🛠 Жергілікті іске қосу

```bash
pip install -r requirements.txt
streamlit run app.py
```
